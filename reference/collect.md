# Retrieve `JSON` data from the browser.

Retrieve `JSON` data from the browser.

## Usage

``` r
# S3 method for class 'tbl_lazy_json'
collect(x, ..., raw = FALSE)
```

## Arguments

- x:

  A `tbl_lazy_json` object.

- ...:

  Unused. Provided for consistency with generic.

- raw:

  A logical, if `TRUE` returns `JSON` as character.
